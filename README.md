# Kalkulator Bunga Sederhana

## Deskripsi
Kalkulator Bunga Sederhana adalah aplikasi web sederhana yang dirancang untuk menghitung bunga sederhana berdasarkan pokok (principal), suku bunga tahunan, dan waktu. Aplikasi ini membantu pengguna memahami dan menghitung bunga yang dihasilkan dari investasi atau pinjaman.

## Rumus Bunga Sederhana
Rumus yang digunakan adalah:
```
I = P * r * t
```
Di mana:
- **I**: Bunga yang dihasilkan
- **P**: Pokok (Principal Amount)
- **r**: Suku bunga tahunan (dalam desimal, misalnya 5% = 0.05)
- **t**: Waktu dalam tahun

Jumlah total yang harus dibayar adalah P + I.

## Teknologi yang Digunakan
- **HTML**: Untuk struktur halaman web.
- **CSS**: Untuk styling dan tata letak responsif.
- **JavaScript**: Untuk logika perhitungan dan interaktivitas.

## Cara Menggunakan
1. Buka file `index.html` di browser web Anda.
2. Masukkan nilai untuk:
   - **Pokok (Principal Amount)**: Jumlah pokok dalam angka.
   - **Suku Bunga Tahunan (Annual Rate)**: Persentase suku bunga (misalnya, 5 untuk 5%).
   - **Waktu (Time in Years)**: Jumlah tahun.
3. Klik tombol "Hitung Bunga".
4. Hasil perhitungan akan ditampilkan di bawah form, termasuk bunga yang dihasilkan dan jumlah total.

Pastikan semua input diisi dengan angka positif yang valid. Jika ada kesalahan, pesan akan muncul.
