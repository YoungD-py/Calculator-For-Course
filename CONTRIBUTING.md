# Panduan Kontribusi

Kami sangat menghargai kontribusi dari komunitas! Semua hal berikut ini sangat kami sambut:

**All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.**

## Cara Berkontribusi
(Anda bisa menambahkan petunjuk singkat tentang Pull Request, misalnya)

1. Fork repositori ini.
2. Buat branch baru untuk fitur Anda.
3. Commit perubahan Anda.
4. Kirimkan Pull Request (PR) ke branch utama.
